The purpose of this tutorial is to give the new users of PHP_CompatInfo an overview of its features and usage patterns.

In brief

There are two types of interface to run a PHP_CompatInfo script dectection: Server API, and CLI. All PHP_CompatInfo features (since 1.4.0) are available on both interface.

Examples that follow, will be explain with SAPI and CLI interface. It's up to you to choose what is the best usage for you.
